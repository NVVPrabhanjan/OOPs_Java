Inheritance is the second pillar of Object-Oriented Programming (OOP).

It allows one class to inherit the properties and methods of another class, promoting code reuse and creating an IS-A relationship.

Syntax
class ChildClass extends ParentClass {

    // Child properties and methods
}
Here:

ParentClass = SuperClass (the one whose properties are inherited)
ChildClass = SubClass (the one that inherits)
Keywords: extends / inherits / uses
